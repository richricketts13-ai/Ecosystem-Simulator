# Ecosystem Simulator

A Java-based ecosystem simulation featuring Grass, Rabbits, and Foxes interacting in a dynamic food chain.

## Features
- Grass growth and reproduction
- Rabbits consume Grass and reproduce
- Foxes hunt Rabbits for survival
- Population tracking each round
- Final simulation summaries
- Object-oriented design using inheritance and polymorphism

## Concepts Practiced
- Inheritance
- Polymorphism
- Lists and iteration
- Simulation loops
- State management
- Ecosystem balancing

## Planned Future Versions
- Organism personalities and traits
- Advanced ecosystem balancing
- Movement and spatial systems
- Expanded statistics and analytics

- ## Version 1 Features
- Basic predator-prey ecosystem
- Rabbits consume Grass
- Foxes hunt Rabbits
- Energy-based survival and reproduction
- Population tracking

- ## Version 2 Features
- Added Rabbit behavioral traits
- Efficient Rabbits gain extra energy from Grass
- Cautious Rabbits conserve movement energy
- Traits are inherited by offspring
- Added probabilistic Fox hunting
- Added trait survival tracking

- ## Version 3 Features
- Added x/y spatial coordinates
- Added organism movement system
- Added bounded world space
- Rabbits only consume nearby Grass
- Foxes only hunt nearby Rabbits
- Ecosystem interactions are now location-based
